Projects

More Than A Number

This website provides a data visualization designed to contextualize a statistic of American adults affected by mental illness. The visualization uses fake images downloaded from This Person Does Not Exist to represent real Americans, which allows me to elicit an emotional response from users without exploiting real people.

Painting With Physics

Painting with Physics is a game I built using Unity and C# with my partner Pauline Cha for our Computer Graphics final. A user clicks on a white cube "wall" and a ball is thrown to that point, with its flight path modeled by the equation of maximum height. When a ball collides with a wall, a paint splatter with the same color as the ball is drawn.

ColorCoding

ColorCoding is a color theme VS Code extension that I created during nCino's Hackathon. It includes 5 custom color themes, and is free to install from the Extension Marketplace. This project was so much fun to make; I enjoyed learning about the extension making process and VS Code’s extensibility.

Link to Extension Manifest

Recognizing ASL Signs Using Machine Learning

I worked with a partner to train a model to distinguish between images of letters in the American Sign Language fingerspelling alphabet. We used the Sign Language MNIST dataset and compared the performance of k-Nearest Neighbors, Random Forest, and Convolutional Neural Networks models. Our best model was the tuned CNN model with an f1 score of 0.95.

Other Work

As part of my coursework at Davidson, I've modeled a Catmull-Rom spline, and used the Phong lighting model to illuminate spheres in orthographic projection and perspective projection. I have also implemented a simulation of the Schelling model, and a dynamic program to print lines of text in a way that achieves the smallest possible "penalty" by minimizing whitespace.
